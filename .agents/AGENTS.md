# Project Rules

- **Implementation Plan Backup**: Whenever an `implementation_plan.md` is created or modified in Planning Mode, save a copy of it in the `implementation` directory under the filename format `plan[YYYYMMDDHHmm].md` (using the current date and time in KST/local time, e.g., `plan202606301314.md`). This must be done sequentially throughout the project.
- **Walkthrough Backup**: Whenever a `walkthrough.md` is created or modified, save a copy of it in the `walkthrough` directory under the filename format `walkthrough[YYYYMMDDHHmm].md` (using the current date and time in KST/local time, e.g., `walkthrough202606301403.md`). This must be done sequentially throughout the project.
